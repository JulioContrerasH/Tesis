\phantomsection
\addcontentsline{toc}{section}{ANEXO A: CÓDIGOS EN PYTHON PARA LA CORRECCIÓN GEOMÉTRICA}
El siguiente código es un conjunto de procedimientos diseñados para la optimización y el análisis de imágenes satelitales. Se apoya en \texttt{rasterio} para el tratamiento de datos geoespaciales, utiliza \texttt{torch} para técnicas de aprendizaje profundo y emplea \texttt{matplotlib} para visualizar los resultados. Esta secuencia de comandos está enfocada en mejorar la calidad espectral y geométrica de las imágenes, y realiza un alineamiento exacto a través del registro de imagen, estableciendo así un enfoque integral para la mejora e interpretación avanzada de datos satelitales.
